# Citator
bookmarklet is the current version
Generate APA 7 or MLA citations from a URL. Three versions for different environments:

| Folder | Best for |
|--------|----------|
| [`powershell/`](powershell/) | Windows — command line or double-click |
| [`html/`](html/) | Any browser — open the file and paste a URL |
| [`bookmarklet/`](bookmarklet/) | Chromebook or any browser — one click on any page, no server needed |

## Quick start

- **Windows desktop** → open `powershell/Get-Citation.bat`
- **Browser** → open `html/Citator.html`
- **Chromebook / any browser** → open `bookmarklet/bookmarklet.html` and follow the setup instructions

See the README in each folder for full details.
